# Landing Transi Base

Estructura base para una nueva landing estatica.

## Estructura

- `index.html`: pagina inicial con placeholder.
- `src/styles.css`: estilos base.
- `src/main.js`: archivo reservado para interacciones futuras.
- `assets/`: imagenes y recursos.
- `.github/workflows/pages.yml`: despliegue en GitHub Pages.

## Como verla

Abre `index.html` directamente en el navegador.
